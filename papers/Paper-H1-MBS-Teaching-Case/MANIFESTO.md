# MANIFESTO: MBS Teaching Case (H1)

> **One-line:** A 90-minute classroom case that teaches organizational diagnosis through Mind-Body-Soul.

---

The MBS framework (Mind = skills/cognition, Body = structure/resources, Soul = purpose/governance) provides a diagnostic lens that students can apply to ANY organization in a single class session. This is not a theory paper. This is a teaching case with instructor notes, discussion questions, and a reusable diagnostic protocol.

**Target Venue:** *Journal of Management Education* / Edited Volume on OD Pedagogy
**Words:** 4,000-5,000  |  **Wave:** 2
