# MANIFESTO: The SPARED Pedagogy Paper

> **One-line:** Dialectical thinking isn't a talent. It's a skill with 5 learnable phases.

---

## The Battle Cry

Critical thinking curricula teach students to EVALUATE arguments. But they never teach them to GENERATE productive disagreement. The result: graduates who can score well on tests but freeze when they must construct a genuine counter-position.

SPARED (SPAR Emergence Design) is a 5-phase pedagogical model for teaching dialectical thinking: SPARK → POSITION → ARGUE → RECONCILE → EMERGE. Each phase corresponds to a Pancha Bootham element (Earth, Water, Fire, Air, Space), grounding abstract cognitive skills in sensory metaphor.

The contribution is not "use debate in class" (that's debate club). The contribution is a **structured emergence design**: a sequence of cognitive states where each phase creates the conditions for the next, and the final phase (EMERGE) produces insight that could not have been predicted from any single phase alone.

## What Gap This Fills

- **Bloom's Taxonomy** describes cognitive levels but not the TRANSITIONS between them
- **Think-Pair-Share** provides structure but no cognitive progression model
- **Socratic Method** uses questioning but has no formal protocol for emergence
- **SPARED** provides a complete pedagogical cycle: from ground (SPARK) to sky (EMERGE)

## The Intended Effect

> *"This is Bloom's Taxonomy with a verb engine. Each phase does something to the student's mind."*
